# 教培学员营收与留存数据分析项目
## 一、项目简介
针对教培机构学员报名、缴费、续费、活动参与数据开展全流程数据分析，完成数据清洗、异常值治理、多维度指标统计、学员价值分层、业务特征工程，并使用Streamlit搭建交互式可视化看板，输出完整分析报告与落地运营建议，为机构精细化运营、业务决策提供数据支撑。

## 二、技术栈
Python（Pandas、NumPy、Matplotlib）、Streamlit、Excel、SQL

## 三、主要分析内容
1. 数据预处理：缺失值填充、重复数据删除、异常数据过滤，输出数据质量报告
2. 多维度分析：课程营收分析、月度营收趋势、学员价值分层、活动效果分析
3. 特征工程：衍生续费潜力、学员活跃度标签，并完成特征有效性验证
4. 可视化应用：静态图表 + 交互式筛选看板
5. 业务输出：梳理现存问题，提供可落地的运营优化方案

## 四、文件说明
- 原始数据：业务源Excel数据表
- 代码文件：Jupyter分析代码、Streamlit交互式看板代码
- 分析报告：完整业务分析报告（含图表、结论、优化建议）
